# Aims and Justification for Website Development

## Website Overview

Notify is a website designed to help users create alarms and reminders in order to keep track of important tasks and events.

The website aims to provide a simple platform where users can quickly create reminders, manage alarms and view upcoming tasks.

Many students and working professionals rely on multiple applications to manage alarms, reminders, calendars and tasks. This can make organisation difficult and may lead to missed deadlines. Also applications can sometimes become unreliable due to updates or device restrictions.

Notify addresses this issue by providing a single platform focused on reminder and alarm management.

# Competitive Analysis

## Competitor: Google Calendar

### Strengths

- Search function allows users to quickly locate events.
- Users can switch between day, week and month views.
- Clear time-based layout.
- Navigation controls allow efficient movement between dates.

### Weaknesses

- Too many icons displayed in one area.
- Interface can appear complex for simple reminder tasks.
- Large empty areas may feel overwhelming to new users.

### Impact on Notify Design

The Notify design focuses on simplicity. Unlike Google Calendar, Notify prioritises reminder creation and management while reducing unnecessary complexity.

### Name
Sarah

### Age
19

University Student

### Goal
Keep track of assignment deadlines.

### Accessibility Need
Mild Dyslexia.

As a student, I want to create reminders for coursework deadlines so that I do not miss submission dates.

- Add reminder
- Set date and time

When I submit a reminder,

Then it should appear in my reminders list.

# Design

The Notify website was designed around the needs of the identified user personas.

The final design includes:

1. Home Page
2. Login Page
3. Create Reminder Page
4. My Reminders Page
5. Contact Page

The design prioritises simplicity, readability and ease of navigation.

# Accessibility and Usability

The Notify design follows key accessibility and usability principles:

### Accessibility Features

- High contrast colour scheme.
- Large clickable buttons.
- Visible labels on forms.
- Readable font sizes.
- Consistent navigation.

### Usability Features

- Simple navigation structure.
- Clear page hierarchy.
- Easy reminder creation process.
- Minimal clutter.
- Efficient reminder management.

---

# Reflection on Design Process

One of the main challenges during the design process was balancing functionality with simplicity.

The competitive analysis showed that applications such as Google Calendar offer many features but can sometimes appear overwhelming for users who only require simple reminder functionality.

As a result, the Notify design was simplified to focus on the most important user actions:

- Creating reminders
- Viewing reminders
- Editing reminders
- Managing alarms

Accessibility considerations such as larger navigation controls, readable text and simplified layouts were used to support users with dyslexia and visual impairments.

The final design provides a user-friendly experience while meeting the needs of students, office workers and older users.
